# bripes — Agência Digital

Site institucional da **bripes**, uma agência de desenvolvimento web e design.

Resumo
- Serviços: desenvolvimento web, landing pages, UI/UX, identidade visual e consultoria digital.
- Tech stack: React, Vite, TypeScript, Tailwind CSS, Framer Motion.

Como rodar localmente
```bash
pnpm install
pnpm dev
# Abra http://localhost:3000
```

Build de produção
```bash
pnpm build
```
